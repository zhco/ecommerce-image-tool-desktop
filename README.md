# 电商图片处理工具 - 桌面版

AI 驱动的电商图片批量处理工具，支持智能抠图、白底填充、20+ 平台尺寸适配。

## 功能

- **AI 智能抠图**：基于 rembg (U2-Net)，一键移除背景
- **白底填充**：自动替换为纯白背景
- **20+ 平台尺寸**：淘宝/拼多多/京东/抖音/Shopee/Amazon 等
- **批量处理**：拖入文件夹，一键处理数百张
- **文字水印**：支持位置/透明度调整
- **色彩增强**：自动优化图片亮度和对比度
- **单张/批量模式**：灵活切换

## 运行

```bash
pip install -r requirements.txt
python main.py
```

## 编译 EXE

```bash
pip install pyinstaller

```
